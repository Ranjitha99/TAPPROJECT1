Tech Glossary

HTML
HTML (HyperText Markup Language) is the foundational language for creating web pages, using tags and elements to structure content like headings, paragraphs, links, and images, telling browsers how to display them.

CSS
CSS (Cascading Style Sheets) is a stylesheet language used to define the presentation and visual appearance of web pages, including colors, fonts, layouts, and spacing.

JAVASCRIPT
JavaScript (JS) is a versatile, high-level programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS.

JAVA
Java is a popular, high-level, object-oriented programming language and computing platform used to build a vast range of applications, from mobile (Android) and web apps to large enterprise systems and games, famous for its "Write Once, Run Anywhere" (WORA) capability, allowing code to run on any device with a Java Virtual Machine (JVM)

REACT
React's primary goal is to make the creation of interactive UIs more efficient through a component-based and declarative approach.
